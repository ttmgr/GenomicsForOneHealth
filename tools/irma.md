# IRMA

## What It Is

[IRMA](https://github.com/peterk87/irma) is a tool used for iterative refinement meta-assembler (flu).

## Description

IRMA (Iterative Refinement Meta-Assembler) functions as an exceptionally robust computational pipeline engineered specifically for the intricate assembly and precise variant calling of highly mutated RNA viruses, like Influenza vectors, natively utilizing short-read and long-read data.

## Basic Usage

```bash
IRMA FLU minion_reads/ my_irma_output/
```

## Usage in This Repository

This tool is integrated into the GenomicsForOneHealth pipelines for robust analysis. 
Please refer to the pipeline specific documentation (`pipelines/`) and the manual execution guides for exact command-line usage and parameters.
