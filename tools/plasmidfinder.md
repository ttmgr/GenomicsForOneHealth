# PlasmidFinder

## What It Is

[PlasmidFinder](https://cge.food.dtu.dk/services/PlasmidFinder/) is a tool used for plasmid identification from assemblies.

## Description

PlasmidFinder is utilized for the identification of plasmids entirely from whole-genome sequencing (WGS) datasets. Concentrating heavily on Enterobacteriaceae and Gram-positive bacteria, it detects specific, scientifically established plasmid replicon sequences.
