# Pling

## What It Is

[Pling](https://github.com/iqbal-lab-org/pling) is a tool used for plasmid graph-based clustering.

## Description

Pling acts as a graph-based sequence clustering tool distinctly engineered for plasmid sequences. Due to the high mobility and pervasive structural variation inherent to plasmids, Pling prioritizes shared gene content and dynamic sequence similarity networks rather than enforcing rigid linear alignments.

## Basic Usage

```bash
pling --input plasmid_list.txt --out pling_output.txt
```

## Usage in This Repository

This tool is integrated into the GenomicsForOneHealth pipelines for robust analysis. 
Please refer to the pipeline specific documentation (`pipelines/`) and the manual execution guides for exact command-line usage and parameters.
