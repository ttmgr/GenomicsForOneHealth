What this tool does

Routes researchers to the closest existing workflow in the collection and shows the main preprocessing defaults and documentation links.

What it does not do

It does not generate full commands, inspect uploaded files, or claim an exact match when the collection does not currently contain one.

Oxford Nanopore quick reference

The selector uses current ONT naming for common entry points such as rapid barcoding, native barcoding, direct RNA, and Dorado FAST / HAC / SUP model tiers.

  • Native Barcoding Kit 24 V14 (SQK-NBD114.24)
