What this tool does

Routes researchers from a generalized sequencing decision tree to the closest existing published workflow example in the collection and returns a practical action sheet: fit assessment, entry points, preprocessing defaults, prerequisites, and documentation links.

What it does not do

It does not inspect uploaded files, replace project-specific judgment, or pretend unsupported scenarios are exact workflow matches.

Oxford Nanopore quick reference

ONT terminology is treated as setup guidance rather than as the first routing layer. Kit/run-mode consequences and Dorado tier notes appear after the main sequencing route is defined.
