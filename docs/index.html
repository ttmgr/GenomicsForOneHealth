What this tool does

Routes researchers to the closest existing workflow in the collection and returns a practical action sheet: fit assessment, entry points, preprocessing defaults, prerequisites, and direct documentation links.

What it does not do

It does not inspect uploaded files, replace project-specific judgment, or pretend unsupported scenarios are exact workflow matches.

Oxford Nanopore quick reference

The selector uses current ONT naming for common entry points such as rapid barcoding, native barcoding, and Dorado FAST / HAC / SUP model tiers.
